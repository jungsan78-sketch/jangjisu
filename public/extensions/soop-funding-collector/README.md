# 장지수용소 SOOP 펀딩 수집기

SOOP 방송 페이지의 화면 채팅을 장지수용소 펀딩 자동메모장으로 보내는 크롬 확장 프로그램입니다.

## 설치

1. 이 폴더를 내려받습니다.
2. Chrome에서 `chrome://extensions` 접속
3. 우측 상단 `개발자 모드` ON
4. `압축해제된 확장 프로그램을 로드` 클릭
5. `public/extensions/soop-funding-collector` 폴더 선택

## 사용

1. 장지수용소 펀딩 자동메모장 페이지를 엽니다.
2. SOOP 방송 페이지를 엽니다.
3. 크롬 확장 아이콘에서 `장지수용소 SOOP 펀딩 수집기` 클릭
4. 열려 있는 SOOP 방송 탭 중 1개 선택
5. 선택한 방송 탭의 채팅만 자동메모장으로 전송됩니다.

## 주의

- 쿠키, 토큰, Authorization 헤더를 수집하지 않습니다.
- 화면에 표시된 채팅 DOM 텍스트만 읽습니다.
- 후원 감지는 기존 Hoxy relay를 유지하고, 후원 후 다음 채팅 매칭만 확장 프로그램으로 보조합니다.
